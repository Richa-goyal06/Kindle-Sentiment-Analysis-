# Kindle-Sentiment-Analysis (Using Natural Language Processing)

This Project is done using Natural Language Processing (NLP) Techniques. Python libraries like Pandas (for Data Cleaning/Handling), NLTK, TextBlob, Scikit-learn (for text preprocessing) and Naive Bayes (for model building)
## Dataset 
- <a href="https://github.com/Richa-goyal06/Bank-loan-Analysis/blob/main/financial_loan.csv">Dataset</a>

## Pipeline
1. Performed data loading, cleaning, and exploratory data analysis using Pandas and NumPy
2. Applied text preprocessing using NLTK (tokenization, stopword removal, stemming/lemmatization) and TextBlob for sentiment polarity extraction
3. Converted text into numerical features using Bag-of-Words and TF-IDF via scikit-learn
4. Split dataset into training and testing sets and trained a Multinomial Naive Bayes classifier
5. Evaluated model performance using accuracy score and confusion matrix and acheived 89% accuracy on test data
